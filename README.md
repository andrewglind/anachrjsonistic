# Anachrjsonistic

The ananchronistic JSON parser - written in C++98 standard C++ that can even be used in Visual C++ 6.0 projects! [Simpleson](https://github.com/gregjesl/simpleson)-like API, extremely lightweight and fast.

## Supported platforms

- [x] Windows 98 with Visual C++ 6.0
- [x] Windows 11 with Visual Studio 2022
- [x] Windows 11 with Visual Studio 2026
